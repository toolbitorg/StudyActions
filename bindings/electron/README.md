# Installation

## Install dependencies


### Ubuntu linux
```shell
  sudo apt install libudev-dev
```

### macOS
Nothing special

### Windows
```shell
  sudo apt install libudev-dev
```

### Raspberry Pi
```shell
  sudo apt install libudev-dev
```

## Install toolbit-lib
```shell
  npm install toolbit-lib
```
